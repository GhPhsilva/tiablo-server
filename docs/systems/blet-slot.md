## Objetivo

Adicionar o slot de item para cintos (belts). No diablo 2 temos um slot para belts, nosso objetivo aqui é adicionar esse slot para o servidor tiablo.

## Pontos importantes

Servidor:
    - precisamos entender quais alterações serão necessárias no client e no servidor
    - precisamos criar uma nova categoria para os items (belt) para especificar onde o item pode ser equipado
    - todos os belts vão ter um attributo que altera a eficacia de poções usadas
    - quando dermos look no belt esse attributo precisa aparecer
    - belts vão ter o attributo defesa

Client:
    - vamos adicionar o slot do belt entre o slot da armadura e da legs
    - o slot do belt não precisa ser do mesmo tamanho dos outros, pode ter a mesma largura e altura menor (caso isso se torne um problema, ele pode ter o mesmo tamanho)
    - para entender o contexto do client pode ler esse arquivo "C:\Users\Pedro\Documents\tiablo\client\CLAUDE.md"
    - vamos precisar criar novas imagens para os icons (belt e belt-blessed) e colocar em C:\Users\Pedro\Documents\tiablo\client\data\images\game\slots
